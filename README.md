# Applied Data Science 1

This repository contains coursework, assignments, and project materials developed for the **Applied Data Science 1** module.  
The focus is on applying core data science techniques — including data wrangling, statistical analysis, visualization, and machine learning — to solve real-world analytical challenges.

## Contents
-  Assignments and reports  
-  Data analysis notebooks  
-  Machine learning experiments  
-  Reflections and learning notes

## Tools and Technologies
Python · Pandas · NumPy · Matplotlib · Scikit-learn · Jupyter Notebooks
